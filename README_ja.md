シェルスクリプトの基礎を学びます。このモジュールでは、最初のスクリプトの作成、変数とユーザー入力、If Else を使用した条件ロジック、For および While を使用したループ、終了コードとエラー処理、スクリプトのスケジュール設定とログについて扱います。実践的なラボと実世界の課題を通じて、これらの不可欠な Linux スキルを習得します。
